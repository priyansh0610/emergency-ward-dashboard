Emergency Ward Command Dashboard - Final Version

How to open
1. Extract the zip file.
2. Open index.html in Chrome, Edge, or Firefox.
3. No installation is required.

Main formulas
1. Patient turnaround time = departureAt − arrivalAt
2. Door to triage = triageAt − arrivalAt
3. Door to clinician = clinicianAt − arrivalAt
4. Bed turnaround time = bed ready time − bed vacated time
5. Risk score = acuity score + waiting time score + alert score + boarding score

Important note
This is a non clinical educational prototype only. All data is fictional demo data and must not be used for real patient care.
